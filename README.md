# Leonardo Benicio profile page
This is my personal website made in jekyll. Feel free to clone and use it!

## Instalation
clone this git repository

```bash
gem install bundler
```
```bash
bundle install
```

## Running
```bash
bundle exec jekyll serve
```

## Build
```bash
bundle exec jekyll build
```

## Unit tests
```bash
bundle exec rake test
```
